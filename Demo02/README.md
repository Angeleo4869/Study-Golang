#### 一、fmt占位符

fmt占位符包括

| 占位符 | 作用           |
| ------ | -------------- |
| %T     | 变量属性       |
| %v     | 变量值         |
| %b     | 二进制         |
| %d     | 十进制         |
| %o     | 八进制         |
| %x     | 十六进制       |
| %s     | 字符串         |
| %#v    | 带引号的字符串 |
| %c     | 字符           |
